# imagesTr 使用说明

这个目录是 nnU-Net raw dataset 里放训练图像的地方。当前仓库里没有真实 NIfTI，只放这份说明，避免误以为数据已经物化完成。

## 这里将来应该放什么

训练机器上，每个病例要放 4 个模态文件，命名格式如下：

| 模态 | 目标命名 |
|---|---|
| t1n | `{nnunet_case_id}_0000.nii.gz` |
| t1c | `{nnunet_case_id}_0001.nii.gz` |
| t2w | `{nnunet_case_id}_0002.nii.gz` |
| t2f | `{nnunet_case_id}_0003.nii.gz` |

例如 `BraTSMET_000001` 应对应：

1. `BraTSMET_000001_0000.nii.gz`
2. `BraTSMET_000001_0001.nii.gz`
3. `BraTSMET_000001_0002.nii.gz`
4. `BraTSMET_000001_0003.nii.gz`

源路径来自 `work_space/G2/results/manifests/nnunet_case_mapping_realonly.csv`。

## 注意事项

1. 不要把真实 NIfTI 提交进 Git。
2. 正式运行 nnU-Net 前，如果训练目录要求只包含 `.nii.gz`，请不要把这份说明文件复制到训练目录，或先删除。
3. synthetic data 不放进这个 real-only Dataset260，应单独建 real+synth dataset。
