# G1 任务分工

## 所属小组

生成模型小组。

## 角色定位

核心开发与迁移。

## 具体任务

1. 深入理解往年冠军方案中的 GAN 代码。
2. 将生成器替换为扩散模型（Diffusion Model）。
3. 优先实现切片式条件潜在扩散模型，支持多模态 MRI（T1、T1Gd、T2、FLAIR）及对应 4 类标签的生成。
4. 在开发环境中调试并训练扩散模型，确保模型能产生合理的高质量合成数据。

## 交付物

- 可运行的扩散模型训练代码，技术路线基于 PyTorch。
- 初步生成样例。

## 协作接口

- 向 G2 提供可复用的扩散模型训练结果、推理方式和生成样例。
- 与分割模型小组对接，支持合成数据有效性验证。
