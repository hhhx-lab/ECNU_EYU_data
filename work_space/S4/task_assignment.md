# S4 任务分工

## 所属小组

分割模型小组。

## 角色定位

SAM 及相关变体负责人。

## 共同前置任务

1. 协助生成模型小组验证合成数据的有效性，对比加入合成数据增强前后的性能提升。
2. 使用统一验证集划分，优先使用 5 折交叉验证的第一折，或使用固定验证集，保证模型对比公平。

## 具体任务

1. 调研适用于脑肿瘤分割的 SAM 变体，例如 Medical SAM、SAM-Med2D、SAM-Adapter。
2. 实现并微调 SAM 模型，重点解决小目标分割问题。
3. 设计自动 prompt 生成策略，例如基于 bounding box 或中心点生成 prompt。

## 交付物

- SAM 模型在验证集上的分割性能报告。
- SAM 变体调研记录。
- 自动 prompt 生成策略说明与实验结果。
