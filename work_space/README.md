# 工作区

本目录按成员或小组划分工作空间：`S1`、`S2`、`S3`、`S4`、`S5`、`G1`、`G2`。

## 使用约定

- 每个成员只修改自己的工作区内容。
- 每个工作区包含 `README.md`，用于记录整体任务和当前进度。
- 每个工作区包含 `task_assignment.md`，用于记录从分工文档拆出的个人任务。
- 建议将代码、数据说明、文档和结果分别放入对应子目录，保持提交内容清晰。

## 子目录模板

- `code/`：代码、脚本、notebook 或实验工程。
- `data/`：个人处理后的数据说明或小规模派生数据。
- `docs/`：个人任务文档、分析记录、方法说明。
- `results/`：图表、实验结果、阶段产出。
