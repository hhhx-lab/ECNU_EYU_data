# S3 任务分工

## 所属小组

分割模型小组。

## 角色定位

Swin UNETR 模型负责人。

## 共同前置任务

1. 协助生成模型小组验证合成数据的有效性，对比加入合成数据增强前后的性能提升。
2. 使用统一验证集划分，优先使用 5 折交叉验证的第一折，或使用固定验证集，保证模型对比公平。

## 具体任务

1. 在真实数据上训练 Swin UNETR。
2. 在真实数据加合成数据上训练 Swin UNETR。
3. 分别完成 5 折训练；可先跑 1 折验证完整流程。
4. 重点监控 Transformer 模型对小病灶（小于 275 mm³）的检测性能。

## 交付物

- Swin UNETR 分割结果。
- 小病灶检测指标，包括 F1-score、AUC 等。
- 真实数据与合成数据增强版本的性能对比。
