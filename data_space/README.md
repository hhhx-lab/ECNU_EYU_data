# 资料区

本目录用于存放项目参考资料、往年材料、2026 年 Task 1 资料、课程学习 PDF、数据集和任务安排等公共材料。

## 目录说明

- `past_articles/`：往年文章、论文、报告等参考资料。
- `past_code/`：往年代码、示例工程、可复用脚本等材料。
- `task1_2026/`：2026 年 Task 1 相关资料。
- `task1_2026/datasets/`：2026 年 Task 1 数据说明、数据契约和数据获取资料；大型原始数据不放入仓库。
- `bios_course_pdfs/`：BIOS 课程学习 PDF。
- `task_assignments/`：成员任务安排、分工表和任务说明。

## 已迁入资料

- `task1_2026/`：BraTS 2026 Task 1 中英文说明、数据集现状、数据契约、Sage 登录资料和 nnU-Net 自定义参考脚本。
- `past_articles/`：2023、2025 往年冠军论文 PDF。
- `past_code/`：2023、2025 往年代码、依赖文件、说明文档和代码解析材料。
- `bios_course_pdfs/`：BIOS AI 课程 C1 到 C11 PDF，以及 C6 扩散模型整理材料。
- `task_assignments/`：原始 Word 分工架构文档。

## 使用约定

- 原始资料尽量保持文件名清晰，建议包含年份、来源和主题。
- 大文件提交前先确认是否适合进入 Git 仓库。
- 对外部资料补充来源说明，便于后续复查。
- 医学影像原始数据、训练集、验证集和压缩包默认不迁入仓库。
